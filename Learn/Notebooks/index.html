Notebooks

This section features short, runnable notebooks designed to help you explore specific ML/AI concepts, tools, and workflows. Many can be launched directly via Google Colab using the “Open in Colab” button included at the top of each page.

These notebooks are meant to be lightweight, interactive learning materials. Some may eventually evolve into full workshops using the Carpentries workbench template, but each stands on its own as a handy introduction to key ML/AI ideas.

Have a notebook that helped you or your team learn something new? We welcome community contributions! Submit your notebook to help others learn and explore ML/AI more effectively.
